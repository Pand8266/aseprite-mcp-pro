## 🖥️ What this is

aseprite-mcp-pro adds AI help to Aseprite through Model Context Protocol. It gives you a set of tools for pixel art, sprites, animation, palettes, export, and Godot use. It is made for people who want help with art tasks inside a pixel art workflow.

This project is for Windows users who want a guided way to work with Aseprite and AI tools from one place. It helps with common sprite tasks, frame work, color work, and file export.
